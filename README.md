# firstassignment
short life poem

행하니까 By taking action, 
처음에는 Initially,
마음으로 In my heart,
표가나게 Noticeably,
달라진다 I became different. 
더행하니 By doing more, 
눈에띄게 Visibly,
달라진다 I became different. 
행할수록 The more I did, 
사람들의 To others, 
눈에띄게 Visibly,
달라진다 I became different.
영과육이 My spirit and body,
눈에띄게 Visibly, 
달라진다 Became different.
